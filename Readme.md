# Ordenar Dados

## Estrutura de dados 

Nesse projeto, foi desenvolvido 4 tipos de ordenacao de dados.

* InsertionSort
* CountingSort
* MergeSort
* QuickSort
  
Esse projeto foi desenvolvido nos codigos clicando nesse [Link](https://github.com/Filiplin/Conversoror.git)

O projeto foi feito na linguagem C
